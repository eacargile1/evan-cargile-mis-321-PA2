# Pet Owner — Booking Confirmation

## Wireframe

```
+------------------------------------------------------------------+
|  LTS - Legendary Training Services                    [Login]    |
+------------------------------------------------------------------+
|  < Back to offerings                                             |
+------------------------------------------------------------------+
|                                                                  |
|  Confirm Your Booking                                            |
|                                                                  |
|  +------------------------------------------------------------+  |
|  |  Individual Session with Jane Doe                           |  |
|  |  Obedience, Puppy Training                                  |  |
|  |                                                             |  |
|  |  Date:  Monday, March 10, 2026                              |  |
|  |  Time:  2:00 PM - 3:00 PM                                   |  |
|  |  Location: [To be confirmed / LTS Training Center]           |  |
|  |                                                             |  |
|  |  Your info:                                                  |  |
|  |  Name:  [________________________]                           |  |
|  |  Email: [________________________]                           |  |
|  |  Phone: [________________________]                           |  |
|  |  Pet:   [________________________]  (optional)              |  |
|  |                                                             |  |
|  |  [Cancel]                              [Confirm Booking]     |  |
|  +------------------------------------------------------------+  |
|                                                                  |
+------------------------------------------------------------------+
```

## Elements

- **Back link**: Return to offerings
- **Summary**: Offering type, trainer, date, time, location
- **Form**: Name, email, phone, pet (optional)
- **Actions**: Cancel, Confirm Booking

## User Flow

1. Arrive from "Book" click on offering
2. Review details
3. Enter contact info (or use logged-in profile)
4. Click Confirm Booking
5. See success message + confirmation details

## Post-Booking

- Success screen: "Booking confirmed. You'll receive a confirmation email."
- Option to add to calendar
